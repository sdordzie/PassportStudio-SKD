PassportStudio Pro — Installable PWA package

What is included
- index.html
- manifest.webmanifest
- sw.js
- icons/icon-192.png
- icons/icon-512.png

Important
- Open this as a hosted web app over HTTPS or localhost.
- For phones and desktop browsers, deploy the whole folder to GitHub Pages, Netlify, Vercel, or a secure server.
- Opening index.html directly as file:// will not allow service worker install features.
- On Android and desktop Chromium browsers, this can install as an app window.
- On iPhone/iPad, it installs as a web app from Safari Add to Home Screen; iOS still treats PWAs differently from native apps.
